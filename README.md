# Font & Design Analyzer

A premium SaaS-style web application to extract typography and design styles from Websites, Images, and PDFs.

## Features
- **Website Analysis**: Extract fonts, colors, and computed styles via Puppeteer.
- **Image Analysis**: OCR and Color Palette extraction.
- **PDF Analysis**: Text and font detection.

## Getting Started

1.  Clone the repository.
2.  Run `npm install`.
3.  Run `npm run dev`.

## Deployment

Refer to `cpanel_deployment_guide.md` or `vercel_deployment_guide.md` in the artifacts for deployment instructions.
